1. Add the complete forder at your servers root directory
2. Go to the url http://slackinvite.com/oneui/admin/createDb.php
3. Provide existing details of server Name, username and password for database,
	automatically it will create required database and redirect you to create admin user
	with username and password for accessing slack users database.

4. you need to get your token from https://api.slack.com/tokens link and you can add that token 
	in the slack_settings table.
5.you are ready to go.
